# Bibliography System - Quick Start Guide

## üöÄ Get Started in 5 Minutes

### Step 1: Enable Bibliography

1. Open your book in the Library
2. Click on **Bibliography Manager** button
3. Go to **Settings** tab
4. Check ‚úÖ **Enable Bibliography**
5. Select your preferred citation style (e.g., APA, MLA, Chicago)

### Step 2: Add Your First Reference

#### Option A: Quick Add (Common Types)
1. Click one of the quick add buttons:
   - **+ Book**
   - **+ Journal Article**
   - **+ Website**
   - **+ Conference Paper**
   - **+ Thesis**

#### Option B: Add Any Type
1. Click **+ Add Reference**
2. Enter the reference type
3. Fill in the form with reference details

#### Example: Adding a Book
```
Title: The Elements of Style
Authors:
  - First: William, Last: Strunk, Suffix: Jr.
  - First: E.B., Last: White
Year: 1999
Publisher: Pearson
Location: New York, NY
Edition: 4th ed.
ISBN: 978-0205309023
```

### Step 3: Insert Citations While Writing

1. Open the **Book Editor**
2. Place cursor where you want to cite
3. Click **üìö Cite** button in toolbar
4. Search for your reference
5. Select it and click **Insert Citation**

**Result:** Citation appears in text like `(Strunk & White, 1999)`

### Step 4: View Your Bibliography

The bibliography automatically appears at the end of your book with all cited references formatted according to your chosen style.

## üìö Common Reference Types

### Book
**Required:** Title, Authors, Year, Publisher
**Optional:** Edition, Volume, ISBN, Location

### Journal Article
**Required:** Title, Authors, Journal Title, Year, Volume, Pages
**Optional:** Issue, DOI, ISSN

### Website
**Required:** Title, URL, Retrieved Date
**Optional:** Authors, Website Name, Publication Date

### Conference Paper
**Required:** Title, Authors, Conference Name, Location, Date
**Optional:** Pages, Proceedings, Publisher

## üí° Pro Tips

### 1. Use Citation Keys
Add a short citation key (e.g., `Smith2020`) for quick reference lookup.

### 2. Tag Your References
Add tags like `methodology`, `theory`, `case-study` to organize references.

### 3. Add Notes
Include personal notes about why you're citing this source or key points.

### 4. Page Numbers
Always add page numbers when citing specific information:
- Direct quotes: Required
- Specific data: Recommended
- General ideas: Optional

### 5. Export Your Bibliography
Save your references for use in other projects:
- **JSON** - Full data with all metadata
- **BibTeX** - For LaTeX documents
- **RIS** - For reference managers

### Scenario: Writing a Non-Fiction Book on Climate Change

#### 1. Setup (1 minute)
- Enable bibliography
- Set style to APA
- Enable in-text citations and bibliography

#### 2. Add References (5 minutes)
Add 5 key sources:

- Title: "The Uninhabitable Earth"
- Author: David Wallace-Wells
- Year: 2019
- Publisher: Tim Duggan Books

**Journal:**
- Title: "Global warming of 1.5¬∞C"
- Authors: IPCC

- Title: "Climate Change: How Do We Know?"
- Website: NASA
- URL: nasa.gov/climate

- Title: "Renewable Energy Solutions"

#### 4. Review Bibliography (2 minutes)

#### 5. Export (1 minute)
Export as PDF with complete bibliography included.

## ‚ùì FAQ

**Q: Can I change citation style after adding references?**
A: Yes! All references automatically reformat when you change styles.

**Q: What if I don't know all the reference details?**
A: Fill in what you have. Only title and authors are required for most types.

**Q: Can I cite the same source multiple times?**
A: Yes! Add one reference, then cite it as many times as needed.

**Q: How do I cite a specific page?**
A: When inserting a citation, add the page number in the "Page Number" field.

**Q: Can I use this for fiction books?**
A: The system is designed for non-fiction, but you can use it for any book that needs references.

**Q: Will references appear in the PDF export?**
A: Yes! Both in-text citations and the complete bibliography are included.
